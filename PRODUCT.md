# Product

## Register

brand

## Users

开发者（个人/团队），拥有数十到数百个 GitHub Stars，日常在 GitHub 上收藏开源项目但难以回顾和利用这些收藏。使用场景：在寻找技术方案时，希望从自己的 Stars 库中快速找到匹配的项目，而非重新搜索全网。

## Product Purpose

GitHub Stars AI Tools (GSAT) 是一款本地优先的桌面应用，将 GitHub Stars 转化为可搜索、可总结、可本地查询的 AI 知识库。官网的目标是让访客在 30 秒内理解产品价值，并产生下载试用的冲动。

成功标志：访客能清晰回答"它能帮我做什么"和"为什么我需要它"，而非仅仅记住功能列表。

## Brand Personality

**聪慧 · 精炼 · 可信赖**

- 聪慧：AI 驱动的智能不是噱头，而是真正理解开发者需求的工具
- 精炼：每一个交互都经过推敲，信息密度高，不做无意义的装饰
- 可信赖：本地优先、隐私安全、开源透明——让人放心把数据交给它

## Anti-references

- **SaaS 模板感**：避免 Vercel/Notion 式的"大标题 + 渐变 + 三列卡片"千篇一律布局
- **过度工程感**：避免 Retool/Supabase 式的"深色终端 + 霓虹色"开发者工具刻板印象
- **空洞营销**：避免只有形容词没有证据的文案，如"革命性 AI 平台"

## Design Principles

1. **内容即设计**：用产品的真实能力和数据说话，而非抽象插图
2. **开发者尊严**：不低估受众智商，文案精准、技术可信，避免花哨的营销话术
3. **本地优先的温度**：通过设计传达"你的数据在你手中"的安全感和掌控感
4. **信息密度与呼吸感平衡**：每屏一个核心信息，但不为留白而留白

## Accessibility & Inclusion

- WCAG AA 级别
- 支持 prefers-reduced-motion
- 色彩对比度 ≥ 4.5:1（正文）/ ≥ 3:1（大文本）
- 键盘可导航
